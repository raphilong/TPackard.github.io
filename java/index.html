Pathfinder

I've been working on A* pathfinding, and this is the result so far. It's very limited, but you can click anywhere and the blue circle will travel there. The green node is where the path starts, and the red node is where it ends. You can click or drag anywhere to place walls. To run the program, simply double-click pathfinder.jar.
