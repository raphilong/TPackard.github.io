Set up a Local Server

February 20, 2014
Local servers are very useful in webdesign for testing your website since they support links to other files in your site much better than when you use file://link-to-file. Fortunately, local servers are very easy to create on Macs with python installed (most newer macs will have python installed by default). To create a local server, simply open up terminal and change directory to the folder where your site is saved. To change directory, type cd directory-name, or ls to list the available directories. To go back out of a directory, simply type cd. Once you have navigated to your site's main directory, type the command python -m SimpleHTTPServer XXXX, where the Xs are your port number. If you don't specify a port, it will default to 8000. To access your new local server, go to http://localhost:XXXX. The server will automatically update any files you change, but you'll still have to reload the page. Remember to leave the terminal window that's running the server open, or else the server will quit. If you wish to quit the server, simply close the window or press control-c. That's it! Now you have a fully functioning local server.
